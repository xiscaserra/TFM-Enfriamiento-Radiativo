# TFM-Enfriamiento-Radiativo
En este repositorio se muestran los códigos usados tanto para la toma de datos como para la representación de éstos.
El archivo Lee-DHT-v3.py corresponde al código introducido en la Raspberry que mide temperatura y humedad cada minuto.
Con archivo reflectancia.py, se han realizado las gráficas necesarias para las medidas de reflectancia hechas con el espectrofotómetro Jasco-V770.
Y con el gráficas_temp.py, se representan las evoluciones de temperatura y humedad además de las distribuciones de las diferencias entre los dos sensores para todos los casos.
Para la realización del código se ha usado ayuda de la inteligencia generativa.
